Scoreboard help

General

A modern browser is recommended, running fullscreen.

Player controls

The scoreboard is intended to be used with external buttons, like a PS2 Buzz! controller (which can be picked up cheaply on eBay).

Players buzz in by pressing buttons mapped to 1, 2, 3 or 4 on the keyboard. The corresponding button numbers on the Buzz! controllers are 0, 5, 10 and 15 – these can be mapped using software like Joystick Mapper, Controllers or Enjoy2 on a Mac, or equivalents on Windows or Linux.

Quizmaster controls

  • Click on a player name to edit.
  • Add and remove points using the red and green buttons.
  • Unlock the players' buzzers by either adding or removing a point, hitting ‘Continue’ or pressing the space bar.
